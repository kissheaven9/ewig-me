# ИИ-биограф ewig.me — прокси к OpenAI (Cloudflare Workers)

Сайт статичный (GitHub Pages), поэтому ключ OpenAI нельзя класть в код — его бы украли.
Ключ живёт в **Cloudflare Worker** (секрет), а сайт зовёт этот воркер. Cloudflare Workers —
бесплатный тариф (100 000 запросов в день), карта не нужна.

## Что понадобится
- Аккаунт **Cloudflare** (бесплатный): https://dash.cloudflare.com/sign-up
- Твой **ключ OpenAI** (у тебя уже есть средства): https://platform.openai.com/api-keys → «Create new secret key», скопировать (`sk-...`).

## Деплой за 5 минут (через сайт Cloudflare, без терминала)
1. Войди в **dash.cloudflare.com** → слева **Workers & Pages** → **Create** → **Create Worker**.
2. Дай имя, напр. `ewig-ai` → **Deploy** (создастся заготовка).
3. **Edit code** → удали весь код, вставь содержимое [`worker.js`](worker.js) из этой папки → **Deploy**.
4. Добавь ключ как секрет: на странице воркера **Settings → Variables and Secrets** →
   **Add** → тип **Secret** → имя `OPENAI_KEY` → значение = твой `sk-...` → **Save and deploy**.
5. Скопируй URL воркера (вида `https://ewig-ai.<твой-субдомен>.workers.dev`).

## Подключение к сайту
Пришли мне этот URL — я впишу его в код (`AI_PROXY_DEFAULT` в `assets/js/app.js`) и перевыложу.
После этого ИИ-биограф на сайте будет генерировать реальные тексты через твой OpenAI.

**Быстрый тест без перевыкладки:** на сайте открой консоль браузера (F12) и выполни
`localStorage.setItem('ewig_ai_proxy','ВСТАВЬ_URL_ВОРКЕРА')`, обнови страницу — ИИ заработает
в твоём браузере. Так можно проверить до того, как я зашью URL в код для всех.

## Как это работает
Сайт → POST на воркер `{name, answers[], lang}` → воркер добавляет ключ и зовёт OpenAI
(`gpt-4o-mini`, дёшево) → возвращает `{formal, personal}` (2 варианта биографии) → сайт показывает
их для выбора. CORS в воркере ограничен доменом сайта (`ALLOWED`), чтобы твой ключ не гоняли чужие.

## Стоимость
`gpt-4o-mini` очень дешёвый: одна генерация ≈ доли цента. Лимит трат можно выставить в
OpenAI (Billing → Usage limits). Если ключ/прокси не заданы — ИИ-биограф на сайте работает
в режиме имитации (без затрат).
